# Rustenburg PGM Production Forecaster (2015–2025 + 2026–2028 Forecast)
**Advanced Data Science Portfolio Project – South African Platinum Mining**

**Business Problem (Rustenburg, SA context)**:  
Rustenburg is the heart of global PGM production. Impala Rustenburg production dropped from 1,450 koz (2015) to 1,275 koz (FY2025) due to Eskom load-shedding, safety stoppages and price swings. Sibanye showed better resilience. This project analyses production performance and forecasts future output using advanced ML.

**Live Demo**: Run `streamlit run reports/streamlit_app.py`

**Skills shown**: Data cleaning, correlations, advanced ML forecasting (Prophet + XGBoost + SHAP + VARMAX), optimisation, interactive dashboard.
